Selamat Datang!

Website ini menggunakan Machine Learning untuk menganalisis fitur lagu Spotify dan memprediksi nilai tertentu menggunakan dua model: Random Forest dan Support Vector Regression (SVR). Pilih model pada menu di samping untuk mulai melakukan prediksi.

Analisis Lagu Spotify Lebih Mudah

Masukkan fitur lagu seperti acousticness, danceability, energy, tempo, dan lainnya. Model akan memprediksi karakteristik atau popularitas lagu berdasarkan dataset Spotify.

Klik salah satu model di sidebar untuk memulai.
